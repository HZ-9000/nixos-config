# nixos-config

Personal NixOS flake with home-manager, built around a shared desktop stack (Hyprland) and per-host hardware configuration.

Format Nix files:

```bash
nix fmt
```

## How it fits together

1. **`flake.nix`** declares inputs (nixpkgs, home-manager, Hyprland, Catppuccin, ‚Ä¶) and delegates to `outputs/`.
2. **`outputs/<arch>/src/<host>.nix`** defines a `nixosConfiguration` by combining:
   - `hosts/<hostname>/` ‚Äî machine-specific hardware and settings
   - `modules/nixos/desktop.nix` ‚Äî shared system + desktop modules
   - optional flake input modules (e.g. nixos-hardware, Catppuccin)
   - `home/hosts/linux/<hostname>.nix` ‚Äî home-manager config for the primary user
3. **`lib/nixosSystem.nix`** wraps `nixpkgs.lib.nixosSystem` and attaches home-manager for `vars.username`.
4. **`home/linux/default.nix`** is the shared user profile imported by every Linux host.
